# Recibos developer

# Back

1- crear el achivo .env para las variables de entorno. Dejo un ejemplo del la coneccion a db de test

```
DATABASE_USER="pablo",
DATABASE_PASS="P4bl0",
DATABASE_HOST="10.5.0.124",
DATABASE_PORT=3306
DATABASE="recibosdb"
```

# Front

1- Instalar dependencias en src/front 
```bash
npm install
```

2- Levantar el servicio con 
```bash
docker-compose up
```
